# INVENTO
📦 INVENTO – Inventory Management System
📌 Overview

INVENTO is a Java-based Inventory Management System designed to help businesses manage products, stock, sales, and receipts efficiently.
The system allows administrators and employees to track inventory levels, manage product information, and maintain transaction records.

This project demonstrates concepts such as:

Object-Oriented Programming (OOP)

Database connectivity

CRUD operations

Modular software design

🚀 Features

🧾 Add and manage products

📦 Track product stock and quantity

🧑 Manage employees and cashier details

🧮 Generate receipts for transactions

🔎 View all inventory records

🗂 Database-based data storage

📊 Efficient inventory monitoring

⚙️ Installation & Setup
1️⃣ Clone the repository
git clone https://github.com/srinivasreddy-rr/INVENTO.git
2️⃣ Open the project

Open the project using IntelliJ IDEA.

3️⃣ Setup the database

Create a database in MySQL:

CREATE DATABASE inventory;

Import the SQL file:

inventory.sql
4️⃣ Configure database connection

Update the database credentials in the project:

username = your_mysql_username
password = your_mysql_password
▶️ Running the Project

Run the main Java file:

Main.java

The system will start and allow you to manage inventory operations.
